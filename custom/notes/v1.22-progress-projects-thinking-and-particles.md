# 版本记录：v1.22-progress-projects-thinking-and-particles

## 3. 问题分析

用户反馈聚焦在三类缺口：

1. 栏目结构：缺少独立进度入口与 thinking 模块。
2. 目标表达：项目区需要明确“大厂实习”目标。
3. 首屏体验：希望更强的技术感动画（类粒子、丝滑）。

## 4. 候选方案比较

方案 A：只改导航文案，不改结构。  
优点：改动小。  
缺点：无法实质解决“回不到进度”和“thinking未施工”。

方案 B：补齐独立栏目 + 粒子动效组件化。  
优点：结构问题和视觉诉求一次解决，且便于后续扩展。  
缺点：新增文件和组件较多。

最终采用方案 B。

## 5. 最终决策

1. 品牌名统一改为 `Yfcccc`。
2. 将深度图栏目单独定义为 `Progress`（锚点 `#progress`）。
3. 新增独立 `Projects` 栏目，承载实习目标与项目目标列表。
4. 新增 `Thinking` 栏目并添加 `test` 占位条目。
5. Hero 改为网站介绍文案，并加入 Canvas 粒子动画背景。
