# 版本记录：v1.16-learning-map-mc-style-static

## 1. 本轮目标

根据用户反馈，将学习地图改为更接近 MC 进度树风格，并满足：

1. 不能拖动。
2. 保留悬停效果。
3. 保留点击详情交互。

## 2. 背景与上下文

本轮先读取：

- `custom/notes/v1.15-learning-map-visual-refresh.md`
- `src/components/LearningMap.tsx`
- `src/components/HomePage.astro`

用户明确表示上一版地图“仍然太差，没达到想要的效果”，并提出“可参考开源图/AI 图”的方向。

## 3. 问题分析

上一版问题在于：

1. 视觉虽提升，但仍偏现代信息图，不够“MC 进度树”气质。
2. 保留了可拖动/可缩放模式，与用户目标冲突。
3. 缺乏明显的像素块状层次与任务树质感。

## 4. 候选方案比较

方案 A：下载外部纹理图直接覆盖背景。

- 优点：改动快。
- 缺点：版权和风格一致性风险；后续维护困难。

方案 B：组件内用原创像素风视觉语言重构。

- 优点：风格可控、无版权依赖、可持续维护。
- 缺点：需要改动节点、边、画布和弹窗样式。

本轮采用方案 B。

## 5. 最终决策

1. 固定地图视图并禁用拖拽缩放。
2. 节点改为硬边方块、像素风配色与块状阴影。
3. 保留悬停摘要和点击详情。
4. 详情弹窗同步改为像素任务面板风格。

## 6. 具体实现

### 6.1 交互限制（满足“不能拖动”）

文件：`src/components/LearningMap.tsx`

- 设置：
  - `nodesDraggable={false}`
  - `panOnDrag={false}`
  - `panOnScroll={false}`
  - `zoomOnScroll={false}`
  - `zoomOnPinch={false}`
  - `zoomOnDoubleClick={false}`
- 缩放范围收窄到 `minZoom=0.95 / maxZoom=1.05`，保证近似固定视图。

### 6.2 MC 风格视觉重构

文件：`src/components/LearningMap.tsx`

- 画布：暗色基底 + 规则像素网格纹理。
- 节点：方形硬边、状态色块、块状投影。
- 提示框：羊皮纸色底 + 粗边框 + 硬阴影。
- 图例：右下角像素化状态说明。

### 6.3 详情面板重构

文件：`src/components/LearningMap.tsx`

- 面板从圆角玻璃风改为像素任务面板。
- 头部状态色、标签和按钮统一到 MC 任务语义。

### 6.4 首页说明同步

文件：`src/components/HomePage.astro`

- 地图说明文案改为：固定视图 + 悬停 + 点击详情，避免误导用户可拖动。

### 6.5 流程记录

文件：`AGENTS.md`

- `最新一期路径` 更新为本轮记录。

## 7. 本轮优点

1. 明确对齐“MC 进度树”视觉方向。
2. 完整满足“不可拖动 + 悬停 + 点击详情”需求。
3. 不依赖外部图片资源，避免版权与风格割裂。

## 8. 本轮缺点与代价

1. 当前仍沿用既有节点坐标，布局逻辑未升级为自动排版。
2. 主题个性更强，若后续做多风格切换需抽离主题变量。
