# 版本记录：v1.20-quant-research-interface-refine

## 1. 本轮目标

按用户最新要求，把 Next 子项目视觉与信息结构从“通用高端官网感”进一步收敛为“量化研究机构界面”：冷静、克制、强控制、低噪、难以模仿。

## 2. 背景与上下文

本轮先读取：

- `AGENTS.md`
- `custom/notes/v1.19-next14-structure-clarity-refactor.md`
- `custom/notes/v1.18-next14-premium-personal-site.md`
- `next-premium-site/` 当前核心代码

延续思路：

1. 继续保留分层架构（config/types/data/layout/sections）。
2. 保留滚动淡入作为唯一主动画。

放弃思路：

1. 放弃 Hero 渐变氛围背景。
2. 放弃项目卡片化与 hover 缩放。
3. 放弃偏营销语气的标题和描述。

## 3. 问题分析

用户目标强调“严肃、理性、高智力控制感”，而上一版仍有以下偏差：

1. Hero 有较强渐变氛围，偏展示型。
2. 项目区采用卡片和缩放，容易接近通用 startup landing 风格。
3. 文案存在“展示叙事”倾向，不够研究日志化。
4. 字体体系存在展示型倾向，不够统一克制。

## 4. 候选方案比较

方案 A：小修样式颜色。  
优点：改动少。  
缺点：风格基调不会根本改变。

方案 B：视觉语言和信息排版同时重构。  
优点：从结构和语气双层对齐“研究机构界面”。  
缺点：改动文件较多。

最终采用方案 B。

## 5. 最终决策

1. 字体统一为 Inter（单一主字体）。
2. 去掉渐变/重阴影/缩放，交互仅保留颜色与透明度微变化。
3. 项目区改为线性研究列表（标题 + 一句话描述）。
4. 方法论改为研究笔记式结构。
5. 版本区改为 changelog 风格日志。
6. About 压缩为高度克制的一句话。

## 6. 具体实现

### 6.1 字体与全局风格收敛

- `next-premium-site/app/layout.tsx`
  - 从 `Sora + IBM Plex Sans` 改为 `Inter` 单字体。
- `next-premium-site/app/globals.css`
  - 统一字体变量与行高。
  - 增加 `research-divider` 与 `research-kicker` 工具类。

### 6.2 主题配置去展示化

- `next-premium-site/tailwind.config.ts`
  - 调整色值（更低饱和蓝色）。
  - 删除渐变动画和强调阴影定义。
  - 新增 `line` 颜色用于结构分隔。

### 6.3 信息架构重写

- `next-premium-site/components/sections/HeroSection.tsx`
  - 改为短句核心声明：`Precision compounds.`
  - 去掉长解释和背景渐变。
  - CTA 改为极简边框按钮。

- `next-premium-site/components/sections/ProjectShowcaseSection.tsx`
  - 标题改为 `Core Work`。
  - 卡片网格改为边线分隔的研究列表。
  - hover 仅保留轻微底色与文字变化。

- `next-premium-site/components/sections/MethodologySection.tsx`
  - 改为“研究笔记式”两列结构。
  - 去掉块状卡片容器。

- `next-premium-site/components/sections/VersionHistorySection.tsx`
  - 标题改为 `Iteration Log`。
  - 时间线收敛为 changelog 视觉。
  - 字段标签统一为研究风格 kicker。

- `next-premium-site/components/sections/AboutSection.tsx`
  - 文案压缩为一条高度概括说明。

### 6.4 数据语气同步

- `next-premium-site/data/projects.ts`
  - 收敛到 `title + description`。
- `next-premium-site/data/methodology.ts`
  - 文本改为约束/验证/审计语气。
- `next-premium-site/data/versionHistory.ts`
  - 改为更冷静的迭代日志表述。
- `next-premium-site/types/content.ts`
  - `ProjectItem` 字段同步简化。

### 6.5 导航语气收敛

- `next-premium-site/config/site.ts`
  - metadata 与导航文案改为 `Research Interface`、`Work/Method/Log`。
- `next-premium-site/components/layout/SiteHeader.tsx`
  - 头部去毛玻璃，改为线性分隔。

## 7. 本轮优点

1. 风格明显从“展示型”收敛为“研究型”。
2. 结构更克制：边线、留白、层级成为主要表达手段。
3. 动效严格遵守“微交互 + 淡入”，无花哨行为。
4. 文案更具系统性与长期迭代语义。

## 8. 本轮缺点与代价

1. 视觉冲击力降低，换来更强专业控制感。
2. 初看信息密度更高，对泛用户吸引力不如营销站。

## 10. 本轮结论

本轮已将站点升级为更接近“量化研究机构内部界面”的表达方式：低噪、高控制、信息主导、迭代可审计，符合用户对“严肃、高智力、难以模仿”的核心要求。
